Dependant Selects jQuery Extension

  • Code

Changes the second select box depending on the value of the first.

Example JS

$(document).ready(function(){
	$('.dependantSelects').dependantSelects();
});

Example JS (with parameters)

$(document).ready(function(){
	$('.dependantSelects').dependantSelects({
		index: 1
	});
});

Parameters

index
Integer - The index to start from for the first select box (default: 0)
firstSelect
Selector - CSS selector for first select wrapper (default: '.firstSelect')
selectResults
Selector - CSS selector for select results select wrapper (default: '.selectResults')
