Input Field default message jQuery Extension

  • Code

Set a default message for input fields which will disappear on focus and reappear if input is blank

Example

$(document).ready(function(){
	$('#searchCriteria').inputMsg({
		message:'Search...'
	});
});

Parameters

message
String - input instruction message (default value: "Text here...").
color
String - message text color (default value: "#ccc").
