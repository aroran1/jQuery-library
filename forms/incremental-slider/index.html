Incremental Slider jQuery Extension

  • Code

Replaces radio buttons or a select box with an incremental slider

Example


Example JS

$(document).ready(function(){
	$('.searchRadius').incrementalSlider();
});

Example JS (with parameters)

$(document).ready(function(){
	$('.searchRadius').incrementalSlider({
		overlayDuration: 3000,
		inputWrapper: '.inputs',
		valueRegion: '.value',
		showOverlay: true
	});
});

Parameters

overlayDuration
Integer/String - duration that overlay appears for (default: 2000)
inputWrapper
String - CSS Selector for input/select wrapper (default: '.inputWrapper')
valueRegion
String - CSS Selector for value region (default: '.valueRegion')
showOverlay
Boolean - Whether to show overlay or not (default: false)
mobile
Boolean - Whether to add mobile device drag events or not (default: false)

Required

  • jquery.event.drag-2.0.js
  • jquery.toucharea.js (if mobile == true)

Events

onmoveincrementalslider
Fires when the slider moves. Index of the radio that is selected is passed as a second parameter
moveincrementalslider
Forces the slider to move. Index of destination to move to is passed as a second parameter
