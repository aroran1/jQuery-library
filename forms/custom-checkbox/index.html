Custom Checkbox jQuery Extension

  • Code

Replaces default checkboxes with custom ones.

Example

Example JS

$(document).ready(function(){
	$('.checkbox').checkbox();
});

Example JS (with parameters)

$(document).ready(function(){
	$('.checkbox').checkbox({
		customCheckbox: 'customCB',
		checked:'selected'
	});
});

Parameters

customCheckbox
String - Class to add to custom checkbox (default: 'customCheckbox')
checked
String - Class to add when the checkbox is checked (default: 'checked')

Events

togglecheckbox
Toggles a checkbox
checkedcheckbox
Fires when a checkbox is checked
uncheckedcheckbox
Fires when a checkbox is unchecked
