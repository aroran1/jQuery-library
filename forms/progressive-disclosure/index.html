Progressive disclosure jQuery Extension

  • Code

Shows and hides content based on selected form elements

Example

Hidden content

Example JS

$(document).ready(function(){
	$('.progressiveDisclosure').progressiveDisclosure({
		rules: [[0, 1, 0], [0, 2]]
	});
});

Example JS (with parameters)

$(document).ready(function(){
	$('.progressiveDisclosure').progressiveDisclosure({
		rules: [[0, 1, 0]],
		question:	'.pdQuestion',
		content:	'.pdContent',
		duration:	'slow'
	});
});

Parameters

rules
2D Array - Each value in each array corresponds to the index of the form element that will allow the content to be shown - add multiple arrays to logical OR the rules. To ignore a particular form element, put the value as -1 (default: [])
question
Selector - Css selector for form element (checkbox, radio, select) (default: '.progressiveDisclosureQuestion')
content
Selector - Css selector for hidden content (default: '.progressiveDisclosureContent')
duration
String/Integer - Duration of slide of hidden content (default: 'normal')
