Custom Select jQuery Extension

  • Code

Replaces default select boxes and list items with custom ones.

Example JS

$(document).ready(function(){
	$('.customSelect').customSelect();
});

Example JS (with parameters)

$(document).ready(function(){
	$('.customSelect').customSelect({
		active: 'on'
	});
});

Parameters

active
String - class added when menu item is selected (default: 'active')
startValue
String - initial selected value that does not appear in list (default: null)
maxHeight
Int - Maximum height for list. If the list height is greater than this value then the scrolling mechanism is implemented (default: 300)
