Custom Radio jQuery Extension

  • Code

Replaces default checkboxes with custom ones.

Example

Example JS

$(document).ready(function(){
	$('.radioGroup').customRadio()
		.bind('checkedradio', function(e, index){
			try{console.log('checked', index);}catch(error){}
		});
});

Example JS (with parameters)

$(document).ready(function(){
	$('.radioGroup').customRadio({
		node: 'rad',
		customRadio: 'customRad',
		checked:'selected'
	});
});

Parameters

node
CSS Selector - CSS selector for radio items (default: '.radio')
elem
String - Element to be appended (default: 'div')
customRadio
String - Class to add to custom radio button (default: 'customRadio')
checked
String - Class to add when the radio button is checked (default: 'checked')

Events

checkradio
Checks a radio button, index is passes as second parameter
checkedradio
Fires when a radio button is checked, index is passes as second parameter
