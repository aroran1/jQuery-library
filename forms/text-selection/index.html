Text Selection jQuery Extension

  • Code

Sets or returns the current text selection. N.B. This extension is currently only designed for use with text inputs.

Example JS

To get the current selection as a string

$("input[type='text']").selection()

Example JS

To set the current selection, supply start an end indexes

$("input[type='text']").selection(0, 10)

Parameters

start
Integer - start of selection.
end
Integer (optional) - end of selection.
