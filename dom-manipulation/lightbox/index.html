Lightbox jQuery Extension

  • Code

Creats a lightbox with an overlay

Example

trigger

Example JS

$('.lightboxTrigger').click(function(e){
	e.preventDefault();
	$lightbox = $.lightbox({
		content: '<p>test content</p>'
	})
	.bind('onclose.lightbox', function(){
		try{console.log('close');}catch(error){}
	});
});

Example JS (with parameters)

$(document).ready(function(){
	$('.lightboxTrigger').click(function(e){
		e.preventDefault();
		$.lightbox({
			content: $('.someContainer').html(),
			className: 'lightboxStyle00',
			windowTop: true
		});
	});
});

Parameters

content
String - String of HTML (default: '')
className
String - Class name to be added to '.lightbox' div (default: '')
windowTop
Boolean - Whether lightbox should appear at top of window (default: false)

Events

closelightbox
Closes the lightbox.
oncloselightbox
Fires when lightbox is closed.
resizelightbox
Trigger to change the width and therefore the position of the lightbox. Pass the width in as a second parameter
