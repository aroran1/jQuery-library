Photo gallery jQuery extension

  • Code

Creates a lightbox to display full-size photos one by one (requires Lightbox)

Example

Example HTML

<div id="demo1" class="demo">
	<a href="images/2726788-1.jpg" class="photo" title="Front and passenger side"><img src="images/2726788-1t.jpg" alt="" /></a>
	<a href="images/2726788-2.jpg" class="photo" title="Front"><img src="images/2726788-2t.jpg" alt="" /></a>
	<a href="images/2726788-3.jpg" class="photo" title="Front"><img src="images/2726788-3t.jpg" alt="" /></a>
	<a href="images/2726788-4.jpg" class="photo" title="Rear"><img src="images/2726788-4t.jpg" alt="" /></a>
	<a href="images/2726788-5.jpg" class="photo" title="Interior"><img src="images/2726788-5t.jpg" alt="" /></a>
	<a href="images/2726788-6.jpg" class="photo" title="Dashboard"><img src="images/2726788-6t.jpg" alt="" /></a>
</div>

Example JS

$(function () {
	$('#demo1').photoGallery();
});

Parameters

itemSelector
jQuery selector statement - a selector statement to match all photo links (default: '.photo')
lightboxClassName
String - Class name to be added to '.lightbox' div (default: 'photoGallery')
includeNav
Boolean - Flag whether to include 'previous' and 'next' links
includeTitle
Boolean - Flag whether to use each link title in the lightbox photo title
