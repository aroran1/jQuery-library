Example JS

$(document).ready(function(){
	$('.tooltip').tooltip();
});

Example JS (with parameters)

$(document).ready(function(){
	$('.tooltip').tooltip({
		selectors: {
			trigger: '.rollOverTrigger'
		},
		classes: {
			overlay: 'rollOver',
			inner1: 'rollOverInner'
		},
		event: 'click',
		active:	'on',
		duration: 'slow'
	});
});

Parameters

selectors
Object literal - Object of CSS selectors (default: {trigger:'.tooltipTrigger',content:'.tooltipContent'})
classes
Object literal - Object of strings (default: classes: {overlay:'tooltipOverlay',inner1:'tooltipInner1',inner2:'tooltipInner2',spout:'spout'})
event
String - Event to show/hide content (default: 'click')
active
String - Class added/removed to trigger (default: 'active')
duration
String/Integer - Duration of fade in/out (default: 'normal')
closeContent
String - Content of close link. If nothing is passed then the close link is not appended (default: '')
className
String - Class name to be added to the overlay (default: '')

Requires

  • jquery.fade-elem.js

Events

toggletooltip
Toggles a node.
onopentooltip
Fires when a node is opened.
onclosetooltip
Fires when a node is closed.
