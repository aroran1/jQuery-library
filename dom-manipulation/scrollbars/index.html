Example JS

$(document).ready(function(){
	$('.scrollable').scrollbars();
});

Example JS (with parameters)

$(document).ready(function(){
	$('.scrollable').scrollbars({
		increment: 10,
		wheelMultiplier: 4,
		minSize: 30,
		horizontalScroll: true,
		active: 'on'
	});
});

Requires

  • jquery.event.drag-2.0.js
  • jquery.mousewheel.js

Parameters

increment
Integer - Number of pixels to move content for one click (default: 20)
wheelMultiplier
Integer - Number of pixels to move content for one scroll wheel pip (default: 2)
minSize
Integer - Minimum pixel size of scrollbar (default: 20)
horizontalScroll
Boolean - Adds a horizontal scrollbar if true. Set to true when content has explicit width (default: false)
active
String - Class to add to arrows when active (default: 'active')
