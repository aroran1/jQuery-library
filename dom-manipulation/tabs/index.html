Example JS (with parameters)

$(document).ready(function(){
	// create tabs object
	$('div.tabs').tabs({
		node:		'.tabNode',
		content:	'.tabContent',
		heading:	'h2',
		selected:	'selected',
		index:		0
	})
	// bind to change event
	.bind('onchangetabs',function(e,tab){
		try{
			console.log(tab);
		}catch(error){}}
	);
});

Parameters

node
Selector - CSS selector for tab nodes (default: '.tabNode')
content
Selector - CSS selector for tab content (default: 'div.tabContent')
heading
Selector - CSS selector for tab heading (default: 'h2')
removeHeading
Boolean - Set false to not remove the heading from the DOM (default: true)
selected
String - Class to be granted to the active tab (default: 'selected')
className
String - Class to be added to the <ul> (default: '')
itemClassName
String - Class to be added to the <li> and <a> plus the index (default: 'tab')
index
Integer - Index of selected tab on load (default: 0)
interval
Integer - Interval between automatic swap of tabs in milliseconds (default: -1)
fadeDuration
Integer - Time taken to fade between states in milliseconds (default: 0)
prepend
Boolean - Prepend or append the tab list (default: true)
hashAware
Boolean - Change tabs based on hash fragment. N.B. Need to include jquery.address-1.3.2.min.js for this to work. (default: false)
sameHeight
Boolean - Make all tabs and wrapper the height of the maximum tab. (default: false)

Requires

  • jquery.fade-elem.js (if fadeDuration > 0)
  • jquery.address-1.4.min.js (if hashAware is set to true)

Events

tabsonchange
Fires when the selected tab is changed. Index of active tab is passed as second parameter.
tabschange
Trigger to change the selected tab. Pass selected index as parameter.
