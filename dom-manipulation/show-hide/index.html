Show/hide jQuery Extension

  • Code

Show/hides content with a trigger.

Example HTML

<p><a href="#" class="showHideAllTrigger">Show/hide all</a></p>
<div class="showHide">
	<a class="showHideTrigger">Show/hide content 1</a>
	<div id="showHideContent">
		<p>some hidden content</p>
	</div>
</div>

Example JS

$('.showHide').showHide()
	.bind('onopenshowhide', function(){
		try{console.log('open');}catch(error){}
	})
	.bind('oncloseshowhide', function(){
		try{console.log('close');}catch(error){}
	});

Parameters

activeClass
String - class applied to trigger when open (default: 'open')
duration
String/Integer - duration of slide (default: 'normal')
event
String - event that triggers show/hide (default: 'click')
trigger
String - css selector (default: '.showHideTrigger')
content
String - css selector (default: '.showHideContent')
container
String - jQuery selector - used to define an inner container such as a tab ID (default: document)

Events

toggleshowhide
Toggles a node.
onopenshowhide
Fires when a node is opened.
oncloseshowhide
Fires when a node is closed.
