Example HTML

<div class="example" id="example1">
	<ul class="tabs">
		<li class="selected"><a href="#">Show all</a></li>
		<li><a href="#" data-group="1">Show group 1, any price</a></li>
		<li><a href="#" data-group="2">Show group 2, any price</a></li>
		<li><a href="#" data-group="2" data-price="200">Show group 2 with price 200</a></li>
		<li><a href="#" data-price="300">Show price 300, any group</a></li>
	</ul>
	<div class="content">
		<p>No group. No price</p>
		<p data-group="1" data-price="100">Group 1. Price 100</p>
		<p data-group="2" data-price="200">Group 2. Price 200</p>
		<p data-price="300">No group. Price 300</p>
		<p data-group="1" data-price="300">Group 1 Price 300</p>
		<p data-group="2" data-price="300">Group 2 Price 300</p>				
	</div>
</div>

Example JS

$(function () {
	$('.example').dataFilter({
		duration: 400,
		filterElements: '.content p',
		html5: true
	});
});

Parameters

duration
String/Integer - duration of filter animation (default: 0)
filterElements
jQuery selector statement - elements to be filtered inside object (default: 'div')
html5
Boolean - Flag whether the doctype is HTML5 and custom data attributes will be used (default: false)
