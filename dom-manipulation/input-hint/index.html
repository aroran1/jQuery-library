Input hint jQuery Extension

  • Code

Add input hint text on the input text fields. It disappears when user clicks in.

Example

Sample form

Example HTML

		<form action="#" method="#">
			<fieldset>
				<legend>Sample form</legend>
				<label for="">Please enter you car registration number.</label>
				<input class="regNumber" type="text" name="registrationNumber" value="" id="regNumber" title="Your Reg" />
			</fieldset>
		</form>

Example JS

$(document).ready(function(){
	$('input[type="text"][title!=""]').hint();
});
