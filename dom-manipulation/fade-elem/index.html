Fade elem jQuery Extension

  • Code

Fades or show/hides elements depending if browser is IE<9.

Example

toggle fade


fade content

Example JS

$(document).ready(function(){
	$('#fadeContent').fadeElem();
});

Example JS (with parameters)

$(document).ready(function(){
	$('#fadeContent').fadeElem({
		fadeSpeed: 250,
		fadeOut: false,
		callback: function(){//do something when finished}
	});
});

Parameters

fadeSpeed
Integer/String - Fade duration (default: 500)
fadeOut
Boolean - Whether to fade out or in (default: true)
callback
Function - Function to run when fade has finished (default: null)
