Example JS

$('.revolver').revolver({
	duration: 500,
	interval: 5000,
	index: 0,
	selectors: {
		window:	'.revolverWindow',
		belt: 	'.revolverBelt',
		item: 	'.revolverItem',
		prev: 	'.revolverPrev',
		next: 	'.revolverNext',
		pager: 	'.revolverPager'
	},
	classes: {
		pagerItem:			'pagerItem',
		pagerItemSelected: 	'pagerItemSelected',
		prevDisabled: 		'prevDisabled',
		nextDisabled: 		'nextDisabled'
	}
});

Parameters

duration
String/Integer - duration of slide (default: 500)
interval
Integer - duration of the pause between slides in milliseconds. If <= 0 then the autoplay functionality is not set (default: 5000)
index
Integer - starting index of visible panel(s) (default: 0)
selectors
Object - Object of CSS Selectors (default: {window:'.revolverWindow',belt:'.revolverBelt',item:'.revolverItem',prev:'.revolverPrev',next:'.revolverNext',pager:'.revolverPager'})
classes
Object - Object of classes (default: {pagerItem:'pagerItem',pagerItemSelected: 'pagerItemSelected',prevDisabled:'prevDisabled',nextDisabled:'nextDisabled'})

Events

onchangerevolver
Fires when the revolver changes to show a different panel.
changerevolver
Changes the revolver, takes the index to change to as a second parameter.
