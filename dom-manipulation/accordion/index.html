Example JS

$(document).ready(function(){
	$('div.accordion').accordion();
});

Example JS (with parameters)

$(document).ready(function(){
	// create accordion object
	$('div.accordion').accordion({
		node:'div.accordionNode',
		heading:'div.accordionHeading',
		content:'div.accordionContent',
		open:'selected',
		update:[
			{selector:'a.update',text:['close','open']}
		]
	})
	.bind('onopenaccordion',function(e,node){try{console.log(node);}catch(error){}})
	.bind('oncloseaccordion',function(e,node){try{console.log(node);}catch(error){}});
});

Parameters

node
Selector - CSS selector for accordion node (default: 'div.node')
heading
Selector - CSS selector for trigger element (default: 'h2')
content
Selector - CSS selector for accordion content (default: 'div.content')
open
String - CSS class which is applied to the node open at any time, also any node with this class hardcoded will be open on load (default: 'div.content')
speed
String/Integer - Speed at which animation occurs, can be any of 'slow', 'normal', 'fast', or any time in ms (default: 'normal')
update
Array - Array containing elements in the header which require their innerHTML to change on open/close. Elements in this array should be objects with a selector property (String) and text property (Array) (default: [])
overlap
Boolean - Flag whether opening a node will overlap on top of an existing open node (i.e. the accordion always stays the same height)
collapsible
Boolean - Flag whether it's possible to close all accordion sections (default: true)
sameHeight
Boolean - Flag whether all nodes should be the same height (default: true)

Events

onopenaccordion
Fires when a node is opened. Event fires on the node.
oncloseaccordion
Fires when a node is closed. Event fires on the accordion. Node closed is passed as second parameter.
openaccordion
Forces a node to open. To be triggered on the node.
closeaccordion
Forces any open nodes to close. To be triggered on the accordion.
