Gets computed CSS styles

Returns the computed style of the first element in a collenction

Example JS

$('div').getCompStyle('zIndex');

Parameters

property
Css property - single css property string in JavaScript style that you want the value of.

Note that different browser compute styles differenty. You will normally want to be specific. E.g. use borderLeftWidth as border may not have a computed value even though it is present in the stylesheet.
