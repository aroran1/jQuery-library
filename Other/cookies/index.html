Cookies

  • Code

This plugin will drop a cookie with a specified name to a page and epires it in a given time

Example JS

$(function () {
	var cookieName = "example";
	$.cookie(cookieName, "", { expires: 7, path: '/' });
});

Parameters

Set a cookie
$.cookie("example", "foo");
Session cookie
$.cookie("example", "foo", { expires: 7 });
cookie available for all paths on your domain
$.cookie("example", "foo", { path: '/' });
cookie limit it to a specific path
$.cookie("example", "foo", { path: '/admin' });
cookie's value
alert( $.cookie("example") );
Delete cookie
$.cookie("example", null);

For more details please see https://github.com/carhartl/jquery-cookie.
