Share jQuery Extension

  • Code

Allows sharing of links on social networking sites

Example

    Example JS

    $('#shareList').share({
	sites: [
		{name: 'facebook', text: false},
		{name: 'twitter', text: false},
		{name: 'linkedIn', text: false},
		{name: 'stumbleUpon', text: false},
		{name: 'digg', text: false},
		{name: 'delicious', text: false},
		{name: 'reddit', text: false},
		{name: 'yahoo', text: false},
		{name: 'buzz', text: false},
		{name: 'email', text: true}
	]
});

    Example JS (with parameters)

    $('#shareList').share({
	elem: 'div',
	url: 'some.url.com',
	title: 'some description text',
	popup: false,
	sites: [
		{name: 'facebook', text: false},
		{name: 'twitter', text: false},
		{name: 'linkedIn', text: false},
		{name: 'stumbleUpon', text: false},
		{name: 'digg', text: false},
		{name: 'delicious', text: false},
		{name: 'reddit', text: false},
		{name: 'yahoo', text: false},
		{name: 'buzz', text: false},
		{name: 'email', text: true}
	]
});

    Parameters

    elem
    String - Tag name that the links should be wrapped in (default: 'li')
    url
    String - URL to share (default: location.href)
    title
    String - Text to share (default: $('title').text())
    popup
    Boolean - Open link in a popup window or not (default: true)
    sites
    Array - Array of objects. E.g. {name: 'facebook', text: false} - name in camelCase (default: [])
